platocdp.policy Installation
----------------------------

To install platocdp.policy using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``platocdp.policy`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        platocdp.policy

* Re-run buildout, e.g. with:

    $ ./bin/buildout
