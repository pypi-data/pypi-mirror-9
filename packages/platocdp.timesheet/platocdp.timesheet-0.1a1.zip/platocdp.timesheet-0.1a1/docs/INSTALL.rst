platocdp.timesheet Installation
-------------------------------

To install platocdp.timesheet using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``platocdp.timesheet`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        platocdp.timesheet

* Re-run buildout, e.g. with:

    $ ./bin/buildout
