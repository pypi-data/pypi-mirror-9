*********************************
Durga: Easy Python REST resources
*********************************

Create easy to use Python objects for REST resources including schema
validation.

Features
--------
* Light weight REST client
* Easy to use API concept consisting of Resource, Collection, Element
  corresponds to `Django`_ ORM with Model, Manager, Instance
* Describe a resource as schema and validate incoming JSON against it
* Compatible with Python 2.7, 3.4 and PyPy

.. _Django: https://www.djangoproject.com/
