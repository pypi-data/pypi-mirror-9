Pyliblinear - liblinear python API¶

pyliblinear is an API for liblinear without using ctypes. It aims for more pythonic access to liblinear’s features, especially featuring lazy evaluations whereever possible.

Development Status¶

Alpha.

The API is not stabilized yet. There also may be a few kinks here and there.
