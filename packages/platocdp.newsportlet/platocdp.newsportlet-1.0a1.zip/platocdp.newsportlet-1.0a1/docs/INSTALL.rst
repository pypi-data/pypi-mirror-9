platocdp.newsportlet Installation
---------------------------------

To install platocdp.newsportlet using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``platocdp.newsportlet`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        platocdp.newsportlet

* Re-run buildout, e.g. with:

    $ ./bin/buildout
